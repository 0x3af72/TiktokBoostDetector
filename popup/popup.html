How it scores
  1. Override: "Promotional content" / isAd → Boosted 99%.
  2. Five signals score −2 (organic) … +2 (boosted): likes/views ×3, saves/views ×3, saves/likes ×2, likes/comments ×1, comments-alive ×1. Saves carry half the weight (hardest to fake).
  3. S = the weighted sum. S>0 → Boosted, S<0 → Organic (S1/S2 drop out when views are hidden).
  4. confidence = 50 + 4·|S| (cap 95%, or 80% with views hidden).
50–60% = a guess · 60–75% = lean · 75%+ = act on it. Works on videos and slideshows. Click any badge or the widget header for the full color-coded signal breakdown.
